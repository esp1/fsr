generate-file

(generate-file out-file endpoint-fn uri)

Evaluates the given endpoint-fn for the given URI, and if the return value is a string or a HTTP 200 Ok response map, will write the response body to out-file. If the return value from the function is anything else, will throw an Exception.

publish-static

(publish-static root-fs-path publish-dir)

Finds all HTTP GET endpoint functions in root-fs-path, invokes them for all known URIs, and publishes their results to publish-dir. Known URIs are either non-parameterized URIs, or URIs tracked with esp1.fsr.track-uri/track-uri and collected in esp1.fsr.track-uri/tracked-uris-atom.
