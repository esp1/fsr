get-root-ns-prefix

(get-root-ns-prefix root-fs-path)

Given a root filesystem path, returns the associated root namespace prefix string. It does this by scanning the root fileystem path for a clojure file, and returning the portion of the namepace corresponding to the filesystem path.

http-endpoint-fn

(http-endpoint-fn method endpoint-meta)

Returns the http endpoint function corresponding to the given request method (e.g. :get, :post, etc) in the given endpoint metadata map.

Will first look for a the given request method in the :endpoint/http map and if found, resolves its matching endpoint function and returns it.

If no matching function is found, will look for a :endpoint/type attribute whose value is a symbol of another namespace in which to recursively look for a :endpoint/http or :endpoint/type key and a matching endpoint function.

If no function is found, returns nil.

ns-endpoint-meta

(ns-endpoint-meta ns-sym)

Returns the endpoint metadata associated with the given namespace symbol, or nil if no such namespace is found. The returned endpoint metadata will contain a :endpoint/ns key whose value is the symbol for this namespace, along with any other metadata associated with the namespace.

resolve-sym

(resolve-sym sym default-ns)

Resolves the given symbol and returns it. If the given symbol is not namespaced, the default namespace is used to resolve the symbol. If the first argument is not a symbol, it is returned as is.

uri->file+params

(uri->file+params uri cwd)

Resolves the given URI to a filesystem file or dir in the given current working directory. If a matching filesystem file or dir is found, returns a vector 2-tuple of the matched file/dir, and a map of matched path parameters. If no matching filesystem file or dir is found, returns nil.
